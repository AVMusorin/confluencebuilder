Transition
==========

reStructuredText supports documents with `transitions`_. Example markup
is as follows:

.. code-block:: none

    A transition (or horizontal rule) can be used to seperate content.

    ----

    This is more content.

Output
------

A transition (or horizontal rule) can be used to seperate content.

----

This is more content.
