Doctest Blocks
==============

reStructuredText provide `doctest blocks`_ are interactive Python sessions
cut-and-pasted into docstrings. Example markup is as follows:

.. code-block:: none

    >>> print 'this is a Doctest block'
    this is a Doctest block

Output
------

>>> print 'this is a Doctest block'
this is a Doctest block
