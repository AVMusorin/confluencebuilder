Highlights
==========

reStructuredText defines a `highlights directive`_. Example markup is as follows:

.. code-block:: none

    .. highlights::

        No matter where you go, there you are.

        -- Buckaroo Banzai

Output
------

.. highlights::

    No matter where you go, there you are.

    -- Buckaroo Banzai
