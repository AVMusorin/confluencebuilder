Pull-quote
==========

reStructuredText defines a `pull-quote directive`_. Example markup is as follows:

.. code-block:: none

    .. pull-quote::

        No matter where you go, there you are.

        -- Buckaroo Banzai

Output
------

.. pull-quote::

    No matter where you go, there you are.

    -- Buckaroo Banzai
