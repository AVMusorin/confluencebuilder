Sphinx
======

This page shows the usage of Sphinx's `table of contents`_. Example markup is
as follows:

.. code-block:: none

    .. toctree::
        :maxdepth: 1

        doca
        docb
        docc

Output
------

.. toctree::
    :caption: Table of Contents
    :maxdepth: 1

    code
    deprecated
    download
    glossary
    hlist
    manpage
    production-list
    seealso
    versionadded
    versionchanged
