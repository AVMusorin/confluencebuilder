Deprecated
==========

Sphinx defines a `deprecated directive`_. Example markup is as follows:

.. code-block:: none

    .. deprecated:: 3.1
        Use :func:`spam` instead.

Output
------

.. deprecated:: 3.1
    Use :func:`spam` instead.
