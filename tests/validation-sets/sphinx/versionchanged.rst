Version changed
===============

Sphinx defines a `versionchanged directive`_. Example markup is as follows:

.. code-block:: none

    .. versionchanged:: 2.5
        The *spam* parameter.

Output
------

.. versionchanged:: 2.5
    The *spam* parameter.
