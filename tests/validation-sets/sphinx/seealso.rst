See also
========

Sphinx defines a `seealso directive`_. Example markup is as follows:

.. code-block:: none

    .. seealso::
        Module `other`
        Documentation of the `other` module.

        `Other Manual <https://www.example.com/manual>`_
        Manual for another reference related to this content.

Output
------

.. seealso::
    Module `other`
    Documentation of the `other` module.

    `Other Manual <https://www.example.com/manual>`_
    Manual for another reference related to this content.
